# ANTIGASPI-Projet

## Installation rapide
Il faudrait tourner le frontend et le backend 
il faudrait créer le fichier dans backend : ".env" qui contient la variable MONGODB_URI
### Backend
```bash
cd backend
npm install
npm run start:dev
```
### frontend

```bash
cd frontend
npm install
ng serve
```
